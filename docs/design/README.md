## Referencias de diseño

- Usa este directorio para mockups, moodboards o capturas de referencia (por ejemplo, `Imagen pegada.png` en tu carpeta de diseños).
- Evita versionar binarios pesados; si el diseño es grande, sube a un storage y coloca aquí el enlace.
- Mantén nombres descriptivos (`landing-restaurante-v1.png`, `paleta-colores.md`) y opcionalmente un `.md` con notas de tipografías/colores.

### Paleta actual de la landing
- Fondos: `#0b1020`, `#0f172a` con acentos ámbar/verde.
- Acentos CTA: ámbar (`#fbbf24`, `#f59e0b`) y verde (`#22c55e`).
- Tipografía base: default sans de Next (personalizable en `frontend_next/src/app/globals.css`).
